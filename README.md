# 🌐 Portfolio Website with Contact Form

This is my personal portfolio website built using HTML, CSS, JavaScript, and Node.js.  
It includes a contact form that sends messages directly to my email using Nodemailer.

---

## 🚀 Features

- ✨ Beautiful and simple portfolio design  
- 📂 Projects section  
- 📧 Contact form (sends email directly)  
- ⚡ Backend using Express.js  
- 🔐 Secure email using environment variables  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Node.js  
- Express.js  
- Nodemailer  
- CORS  
- Dotenv
